# AQI Forecasting using RNN

## Project Overview
This project predicts Air Quality Index (AQI) values using Recurrent Neural Networks (RNN) based on historical EPA air quality data.

## Objectives
- Analyze AQI trends
- Build a forecasting model using RNN
- Predict future AQI values
- Visualize prediction results

## Technologies Used
- Python
- Pandas
- NumPy
- TensorFlow/Keras
- Matplotlib
- Jupyter Notebook

## Dataset
EPA Daily AQI Dataset (2022–2024)

## Results
The RNN model was trained on historical AQI data to forecast future air quality trends and evaluate prediction performance.
